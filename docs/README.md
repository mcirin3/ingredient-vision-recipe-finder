# Ingredient Vision Recipe Finder

An AI-powered web application that converts images of available ingredients into ranked, cookable recipe recommendations.

## Tech Stack
- Frontend: Next.js, Tailwind CSS
- Backend: FastAPI
- AI: OpenAI Vision API, GPT
- Recipes: Spoonacular API
- Storage: AWS S3
